### 3. Frontend Setup

Open a new terminal and navigate to the frontend directory:
```bash
cd ../frontend
```

Install dependencies:
```bash
npm install
```

Start the Frontend Development Server:
```bash
npm run dev
```
The frontend will typically run at `http://localhost:5173/`.
